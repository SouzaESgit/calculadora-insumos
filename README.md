# calculadora-insumos
No dia-a-dia é de grande imporancia manter seus dados organizados, através do uso da calculadora tarefas que demandam tempo e que são feitas manualmente podem acarretar em falhas que prejudicará a sua organização, então a calculadora busca tornar uma parte massante em algo mais dinamico e didatico ao seu cotidiano. 
A calculadora de insumos desenvolvidada neste trabalho tem como objetivo efetivar calculos dinâmicos dentro da área agricula, com capacidade e suporte para produção de milho e soja, disponibilidade e quantidade de adubos, juntamente com calculos para para determinação de hectares lineares 
Os valores utilizados nesta atividade são valores médios e bases aproximadas para o maior realismo com as diversidades do campo.
