# calculadora-insumos/Manual de uso 
No dia-a-dia é de grande importância manter seus dados organizados, através do uso da calculadora estas tarefas que demandam tempo e que são feitas manualmente podem acarretar em falhas que prejudica a organização, então, a calculadora busca tornar uma parte massante em algo mais dinâmico e didático ao seu cotidiano. 

A calculadora de insumos desenvolvidada neste trabalho tem como objetivo efetivar calculos dinâmicos dentro da área agricultura, com capacidade e suporte para produção de milho e soja, disponibilidade e quantidade de adubos para ambas monocultura, juntamente com cálculos para para determinação de hectares lineares.

Os valores utilizados nesta atividade são valores médios e bases aproximadas para o maior realismo com as diversidades do campo.

Pré-requistos:

Python v.3.13
https://www.python.org/downloads/
IDLE python v.3.13 (64-bit)

Pode ser acessado via VS code.
https://code.visualstudio.com/download


Como utilizar o programa:
1- Acessar o link abaixo para realizar o download dos arquivos.
2- https://github.com/SouzaESgit/calculadora-insumos
3- Após abrir o link, clique na opção "<>Code" e "Download ZIP"
4- Extraia o arquivo, apareçera uma pasta com o nome "calculadora-insumos-main"
5 - Dê um duplo clique na pasta, passe o mouse no arquivo "calculadora", clique com o botão direito do mouse
6 - Abrir com - selecionar a opção "python" ou "Visual estudio code"
7 - Se abrir via "python" o sistema já poderá ser executado normalmente. 
8 - Caso utilize o VScode, no canto superior direito do programa tem um simbolo de executar o programa onde surgirá um terminal para realizar os calculos.
